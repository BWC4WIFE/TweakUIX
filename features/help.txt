Info about a tweak: Move the cursor over a tweak to view a brief explanation.
                                          
Check (Button): Determines which tweaks are enabled and configured on your system or not. NO system changes are done yet! 
                                           
Apply (Button): This will enable all selected settings.
                                           
Undo (Button): This will restore the default Windows settings.
                                           
Issues needs to be fixed (State): This indicates that the Windows settings are in place.
                                           
Issues already fixed (State): This indicates that the recommend settings are in place.
