Info about a tweak: Move the cursor over a tweak to view a brief explanation.
                                          
Check (Button): Determines which tweaks are enabled and configured on your system or not. NO system changes are done yet! 
                                           
Apply (Button): This will enable all selected tweaks.
                                           
Undo (Button): This will restore the default Windows tweaks.
                                           
Issues needs to be fixed (State): This indicates that the Windows tweaks are in place.
                                           
Issues already fixed (State): This indicates that the recommend tweaks are in place.

Load template...: This will load a preset file with recommended tweaks.
.
Save template: This allows you to set your favorite tweaks and save them to a local file.
